# SkyCast AI — Modern Weather Dashboard 🛰️

A visually stunning, AI-powered weather application that combines a futuristic "Apple-meets-Tesla" aesthetic with real-time meteorological intelligence.

![SkyCast AI Interface](file:///C:/Users/shara/.gemini/antigravity/brain/b64f5b4c-f5b5-4d2a-ae01-d902aa78a744/media__1773160310325.png)

## ✨ Features

- **Ultra-Premium Aurora Background**: A multi-layered, smooth-flowing animated sky with drifting clouds, depth-aware parallax, and atmospheric AI particles.
- **AI-Powered Accuracy**: Utilizes **Gemini 1.5 Flash** integrated with **Google Search Grounding** to provide live, up-to-the-minute weather data instead of stale pre-trained info.
- **Contextual AI Insights**: Smart tips and specialized music atmosphere genres (e.g., Chill Lofi, Cyberpunk Beats) that adapt dynamically to current conditions.
- **Dynamic Weather Layers**: Real-time visual overlays for Rain, Snow, and Storms (including soft lightning flashes).
- **Compact Dashboard UI**: A "sexy", high-end design optimized for clarity and visual impact.
- **AI Voice Summaries**: Instant audio weather reports at the click of a button.

## 🛠️ Tech Stack

- **Framework**: React 19 + Vite
- **Animations**: Framer Motion
- **Intelligence**: Google Gemini AI (Generative AI SDK)
- **Data Grounding**: Google Search Tool
- **Icons**: Lucide React
- **Styling**: Modern CSS3 (Glassmorphism & Flexbox)

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/sharath-444/AI-Modern-Weather.git
   cd AI-Modern-Weather
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment Variables:**
   Create a `.env.local` file and add your Google AI Studio API key:
   ```env
   VITE_GEMINI_API_KEY=your_key_here
   ```

4. **Launch the app:**
   ```bash
   npm run dev
   ```

## 🧠 AI Integration

Unlike standard weather apps that hit a REST API, SkyCast AI uses a LLM-based reasoning engine. It searches the live web for the exact coordinates and conditions of your city, processes the raw data, and generates a poetic description along with intelligent lifestyle tips.
